# Global Superstore Sales Dashboard (Power BI)

## Project Overview
This project analyzes the Global Superstore dataset using Microsoft Power BI to generate insights into sales, profit, and customer trends across different regions and product categories.

##Problem Statement
Businesses that operate across multiple regions often generate large volumes of sales data. However, analyzing this data manually makes it difficult to identify key trends, profitable product categories, regional performance, and customer purchasing behavior.

The Global Superstore dataset contains detailed information about orders, sales, profit, shipping modes, customer segments, and regions. Without proper data visualization and analysis, it is challenging for decision-makers to quickly understand business performance and make data-driven decisions.

Therefore, the objective of this project is to develop an interactive dashboard using Microsoft Power BI that transforms raw sales data into meaningful visual insights. The dashboard will help analyze sales performance, profit trends, regional distribution, and product category performance, enabling better strategic decision-making.
## Objective
The objective of this project is to build an interactive dashboard that helps businesses analyze:

• Sales performance
• Profit trends
• Regional performance
• Category-wise sales
• Customer segments

## Dataset
Global Superstore Dataset containing:

• Orders
• Sales
• Profit
• Ship mode
• Customer segment
• Region and country

## Tools Used
- Microsoft Power BI
- Excel
- Data Visualization
- DAX Measures

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders
- Sales by Region
- Profit by Category
- Sales Trend by Year
- Top 10 Customers

## Key Insights
- Technology category generates highest profit
- Western region has highest sales
- Standard shipping mode is most used
